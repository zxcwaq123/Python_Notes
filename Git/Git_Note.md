# Git工作区域和文件状态

## 1. 工作区域

`工作区（Working Directory）`

* .git 所在目录
    
`暂存区（Staging Area/Index）`

* .git/index

`本地仓库（Local Repository）`

* .git/objects

## 2. 文件状态

`未跟踪（Untrack）`

`未修改（Unmodified）`

`已修改（Modified）`

`已暂存（Staged）`
